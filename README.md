
## Description

PAYMENT API 

## Installation

```bash
$ npm install
add .env with below parameters 
1. MONGO_URI=
2. SERVER_PORT=
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
